简介

js由3部分组成：

  1. ECMAScript
  2. BOM 整个浏览器
  3. DOM

语法

=== 运算符需要在类型和值两方面同时相等

use strict” 定义 JavaScript 代码应该以“严格模式”执行

对象是易变的：它们通过引用来寻址，而非值

for…in 语句遍历对象的属性

delete 关键词会同时删除属性的值和属性本身

使用set和get关键字设置属性的值或读取，读取是不需要()，更简洁。

运算符

  • typeof 返回变量的类型
  • instanceof 返回true，如果对象是对象类型的实例
    以上两个运算符不用括号 直接空空格分割

位运算符处理 32 位数。该运算中的任何数值运算数都会被转换为 32 位的数。结果会被转换回 JavaScript 数。
typeof 运算符对数组返回 “object”，因为在 JavaScript 中数组属于对象。

箭头函数

引入箭头函数有两个方面的作用：更简短的函数并且不绑定this。

(param1, param2, …, paramN) => { statements }

多条语句加花括号，需要手动返回

(param1, param2, …, paramN) => expression

生成器函数 generator

yield关键字暂停函数并返回

生成器函数可以使用for of循环

方法：

  • next() 如果给next方法传参数， 那么这个参数将会作为上一次yield语句的返回值 next()返回一个迭代器对象，done表示是否执行完函数，value为值

类

ECMA6 定义类的方法

js类

使用 super.method(…) 调用父方法。 使用 super(…) 调用父构造函数

es6的class 使用extends继承

继承类中的构造函数必须调用 super(…),(!)并且在使用 this 之前执行它。

当访问一个对象的属性时，首先在这个对象奔上上进行查找。如果没找到，会继续在原型上查到，也就是__proto__指向的那个对象

apply：调用一个对象的一个方法，用另一个对象替换当前对象。例如：B.apply(A, arguments);即A对象应用B对象的方法。

call：调用一个对象的一个方法，用另一个对象替换当前对象。例如：B.call(A, args1,args2);即A对象调用B对象的方法

ES5要继承对象本身和对象的prototype时需要混合继承，使用基类的apply方法时派生类继承基类本身的方法和属性，使用Object.create继承prototype的方法和属性，使用ES6时可以直接继承

数据类型

JavaScript 数据类型JavaScript 中有五种可包含值的数据类型：

  • 字符串（string）
  • 数字（number）
  • 布尔（boolean）
  • 对象（object）
  • 函数（function）

有三种对象类型：

  • 对象（Object）
  • 日期（Date）
  • 数组（Array）
    同时有两种不能包含值的数据类型：
  • null
  • undefined

对象

window

  • window.location.assign() 方法加载新文档。
  • window.open(“about:blank”,”_self”).close() 关闭当前页面 只有open方法打开的才能用close
  • setTimeout(function, milliseconds) 在等待指定的毫秒数后执行函数。
  • setInterval(function, milliseconds) 等同于 setTimeout()，但持续重复执行该函数。
  • clearTimeout()
  • clearInterval()停止执行

Math

Math对象没有构造函数。方法和属性是静态的

  • Math.random() 总是返回小于 1 的数。
  • Math.random() 与 Math.floor() 一起使用用于返回随机整数。
  • Math的random()方法返回0-1之间的随机数，可以用0.5减去随机数进行随机排序。

    arguments

    arguments 对象包含函数调用时使用的参数数组。js变参可以直接传，函数定义内使用arguments关键字和[]索引参数如果调用参数时省略了参数（少于被声明的数量），则丢失的值被设置为：undefined。

this

在 JavaScript 中，被称为 this 的事物，指的是“拥有”当前代码的对象。当不带拥有者对象调用对象时，this 的值成为全局对象。如果一个函数不是 JavaScript 对象的方法，那么它就是全局对象的函数

箭头函数的this

箭头函数的this指向定义时的作用域而不是调用时， 箭头函数不会创建自己的this,它只会从自己的作用域链的上一层继承this
代码没有在任何函数中执行，而是在全局作用域中执行时，this的值就是global对象，对于浏览器来说，this就是window。

正则表达式

语法

可以通过new RegExp()构造函数给正则表达式添加变量，注意字符串里要使用\\转义

修饰符

i 执行对大小写不敏感的匹配。

g 执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）。每次匹配使用当前正则对象的lastIndex属性的值作为在目标字符串中开始查找的起始位置。lastIndex属性的初始值为0，找到匹配的项后lastIndex的值被重置为匹配内容的下一个字符在字符串中的位置索引，用来标识下次执行匹配时开始查找的位置，如果不加g，则每次都从0开始索引

m 执行多行匹配。

方括号

[]表示可以出现的范围,^在[]中查找任何不在方括号之间的字符表示，例如[^abc]|或()分组，将内容作为一个整体进行匹配, 正则表达式中每个小括号内的部分表达式就是一个子表达式。
RegExp.$1...$9属性的值为String类型，返回上一次正则表达式匹配中，第n个子表达式所匹配的文本（只是分组里的文本，不包括定义量词）。

表达式 描述
[abc] 查找方括号之间的任何字符。
[^abc] 查找任何不在方括号之间的字符。
[0-9] 查找任何从 0 至 9 的数字。
[a-z] 查找任何从小写 a 到小写 z 的字符。
[A-Z] 查找任何从大写 A 到大写 Z 的字符。
[A-z] 查找任何从大写 A 到小写 z 的字符。
[adgk] 查找给定集合内的任何字符。
[^adgk] 查找给定集合外的任何字符。
(red|blue|green) 查找任何指定的选项。

Quantifiers 定义量词

n+ 匹配任何包含至少一个 n 的字符串。
n* 匹配任何包含零个或多个 n 的字符串。
n? 匹配任何包含零个或一个 n 的字符串。
n{X} 匹配包含 X 个 n 的序列的字符串。
n{X,Y} 匹配包含 X 至 Y 个 n 的序列的字符串
。
n{X,} 匹配包含至少 X 个 n 的序
列的字符串。
n$ 匹配任何结尾为 n 的字符串。
^n 匹配任何开头为 n 的字符串。^符号在[]内是取反的意
?=n 匹配任何其后紧接指定字符串 n 的字符串。
?!n 匹配任何其后没有紧接指定字符串 n 的字符串。

注意事项

微信内置浏览器有缓存，如果js文件名不变的话是不会重新加载的

背景色用内联样式，否则获取不到

bj.style这个方法只能获取写在html标签的写在style属性中的值

window.innerWidth/innerHeight无法获取手机端大小

可以使用document.body.clientWidth获取，但是body默认高度是0，可以将body继承window的高度后再获取

setInterval只能传递函数指针，否则第二次会将函数的返回值传递进去，可以使用一个匿名函数执行带参数的函数

在函数内使用getComputedStyle获取一个参数字符串时使用中括号[]

img默认是行内元素，使用display:block可以设置成块级元素并且消除默认的3px空白

offsetWidth实际获取的是盒模型(width+border + padding)

js文件在本地运行和服务器上路径不一样，本地以html为基准，服务器上以js文件为基准
