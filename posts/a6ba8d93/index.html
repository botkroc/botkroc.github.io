DOS命令

  • 参数
  • 网络相关

基础命令

  • type

    • 显示文本 管道符 | + more 分页回显

  • echo

    • 回显，默认输入到屏幕 使用>符号(覆盖)输出到指定文件

    • echo off

      • 不回显之后的执行过程
      • @echo off 不回显当前行

    • echo.

      输出空行

  • cls(clean screen)

    • 清屏

  • rd(remove directory)

    • 删除目录命令 /s 除目录本身外，还将删除bai指定目录下的所有子目录和文件。用于删除目录树。 /q 安静模式，删除目录树时不要求确认，以下命令删除所有目录树

  • color

  • title

    修改标题

  • cd

  • del 删文件

    • del . /s/q 删除所有文件

  • copy

    • 复制文件

  • move

    • 移动文件 用法参考copy

  • ren (rename)

  • md (make directory)

    • 创建一个子目录

  • attrib (attribute)

    • 为文件或文件夹添加属性

  • dir

    • dir /a 浏览所有文件（包括隐藏和系统文件）

  • fsutil file createnew

    • assoc

  • shutdown

  • logoff 注销 同 shutdown -l

  • taskkill

    按照进程ID或印象名称终止任务，详情taskkill /?查看

  • at

    计划任务

参数

  • >>为追加，>为覆盖

  • /q quiet 安静模式 /s 递增删除

网络相关

ping IP -i size 更改数据包TTL时间，能够跟踪数据包途径的路由器

ping -l 2000 发送2000字节的数据包到目标主机

ping -t 一直ping

ping -n 5 指定ping5个包

ipconfig 查看基本IP信息

ipconfig /all 查看详细IP信息

ipconfig /renew
重新请求ip地址

ipconfig /release
释放租约

ipconfig /flushdns
清除dns缓存

netstat 查看网络连接 或已建立侦听的端口，| find”ESTABLISHED”

telnet 测试远程计算机某个端口是否打开

nslookup 查看dns

arp -a 查看arp缓存

arp -d 清除ARP缓存（ARP缓存表储存在内存中）

arp -sARP绑定

route print 查看路由表

  • 远程桌面
    • mstsc

用户相关

  • net user

    查看用户

    • net user /del

      删除用户

  • net user administrator a1!重设密码

  • net user han a1! /add 添加用户

  • net user han /active:yes/no 激活或禁用账户

  • whoami /user 查看当前用户和sid

组管理

  • net localgroup

    查看组

共享文件夹

  • net share

    • 共享名后加$隐藏

  • net share name /del 删除共享

被访问共享文件夹的主机必须设置管理员帐号密码

  • net use f: \10.1.1.1 123.com /user:administrator

    映射网络驱动器
