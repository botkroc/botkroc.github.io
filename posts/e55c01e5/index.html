常见端口号

发表于2020-12-24|更新于2020-12-24|计算机网络
名称/服务 协议 端口号
ssh TCP 22
telnet TCP 23
RDP(remote desktop protocal) TCP 3389
FTP TCP 21
SQL TCP 1433
DNS UDP(也可以用TCP) 53
共享服务,SMB TCP，容易被攻击，最好禁用 445
POP3 TCP 110
DHCP UDP DHCP有3个端口，其中UDP67和UDP68为正常的DHCP服务端口，分别作为DHCP Server和DHCP Client的服务端口；546号端口用于版DHCPv6 Client，而不用于DHCPv4，是为DHCP failover服务
SQL Server TCP
Oracle TCP 1521
MySQL TCP 3306
