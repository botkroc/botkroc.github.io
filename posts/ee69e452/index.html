CSS简介

Cascading Style Sheets是一种用来表现HTML（标准通用标记语言的一个应用）或XML（标准通用标记语言的一个子集）等文件样式的计算机语言。CSS不仅可以静态地修饰网页，还可以配合各种脚本语言动态地对网页各元素进行格式化。

CSS 属性

CSS3 动画属性 （Animation）

  • @keyframes 规定动画。
  • animation 所有动画属性的简写属性，除了 * animation-play-state 属性。
  • animation-name 规定 @keyframes 动画的名称。
  • animation-duration 规定动画完成一个周期所花费的秒或毫秒。
  • animation-timing-function 规定动画的速度曲线。
  • animation-delay 规定动画何时开始。
  • animation-iteration-count 规定动画被播放的次数。
  • animation-direction 规定动画是否在下一周期逆向地播放。
  • animation-play-state 规定动画是否正在运行或暂停。
  • animation-fill-mode 规定对象动画时间之外的状态。

  • background-color 设置元素的背景颜色。

  • background-position 设置背景图像的开始位置。
  • background-repeat 设置是否及如何重复背景图像。

    • content-box

  • overflow-x 如果内容溢出了元素内容区域，是否对内容的左/右边缘进行裁剪。
  • overflow-y 如果内容溢出了元素内容区域，是否对内容的上/下边缘进行裁剪。

浮动和定位

浮动

说明

当元素设置float后，会从流中脱离，排在下面的元素会往上浮动，和设置浮动的元素排成一列
浮动元素会脱离网页文档，与其他元素重叠，但是，不会与文字元素重叠

浮动元素不占据空间，其父元素如果不设置大小并不会被浮动元素撑开，或者使用overflow:auto使父元素自动调整大小（只有高度会自动适应子元素，宽度会与网页宽度一致）

浮动元素在排列时，只参考前一个元素的位置即可（如果有多个浮动元素，大小超过父元素需要换行，此时只需参考浮空元素的前一元素）。

浮动元素的width不跟父元素相同 而是根据自身的内容决定

浮动元素在父元素放不下时会自动换行排列

可以将想要横向排列的元素设置到div里，这样最后一个元素就不会和div的下一个元素产生浮动影响。
浮动只影响后面的元素

清除浮动

clear:left/right/both可以清除当前元素不受浮动元素的影响

clear只会操作块元素

:after和:before是css3中的伪类元素。用法是像元素的前或者后(是在元素中的内容之后)插入元素
content里面是内容，下面接属性，和css写法一样，可以用于清除嵌套浮动的影响,
原理是浮动元素不会撑开父元素，而在浮动元素后面添加一个新元素并clear后父元素就可以被撑开了，使用伪类可以避免添加一个空元素
css3为了区分伪类和伪元素，伪元素采用双冒号写法。

after伪类是内联的，只有block元素才能清除浮动，而此方法只能消除子元素对父元素的浮动影响，
两个同级元素可以使用BFC规范使其消除浮动影响

定位

position

绝对定位absolute的参照对象是“离它最近的已定位的祖先元素”，例如，将div设置为relative，此时它的子元素的absolute定位的参照对象是此div， 相对定位的特性是虽然它定位了，就算给了偏移量它离开了原来的地方，但是它原来占的地方也不会让出来的

    flex弹性盒模型（flex布局）

    flex布局，可以简便完整，响应式地实现各种页面布局，垂直居中很方便

作用在flex容器上 作用在flex子项上
flex-direction order

取值： row、row-reverse、column、column-reverse

flex-wrap flex-grow

取值：nowrap(默认值，单行显示不换行)、wrap(宽度不足换行显示)、wrap-reverse(宽度不足换行显示，但是从下往上开始)

flex-flow flex-shrink
direction和wrap的复合写法
justify-content flex-basis

决定主轴方向上子项的对其和分布方式,对单行不起作用
取值：

  • flex-start:默认值，表现为起始位置对齐。
  • flex-end:表现为结束位置对齐
  • cneter:表现为居中对齐
  • space-between：表现为两端对齐，between是中间的意思，意思是多余的空白间距只在元素中间区域分配
  • space-around:around是环绕的意思，意思是每个flex子项两侧都环绕互不干扰的等宽的空白间距，
    最终视觉上边缘两侧的空白只有中间空白宽度一半。
  • space-evenly:evenly是匀称、平等的意思，也就是视觉上，每个flex子项两侧空白间距完全相等
align-items flex

align-items中的items指的就是flex子项们，因此align-items指的就是flex子项们相对于flex容器在”侧轴”方向上的对齐方式
取值：

  • stretch:默认值，flex子项拉伸
  • flex-start:表现在容器顶部对齐
  • flex-end:表现为容器底部对齐
  • center:表现在垂直居中对齐
align-content align-self

可以看成和justify-content是相似且对立的属性，如果所有flex子项只有一行，则align-content属性是没有任何效果的
取值：

  • stretch:默认值，每一行flex子元素都等比拉伸
  • flex-start:表现为起始位置对齐
  • flex-end:表现为结束位置对齐
  • cneter:表现为居中对齐
  • space-between:表现为两端对齐
  • space-around:每一行元素上下都享有独立不重叠的空白空间
  • space-evenly:每一行元素都完全上下等分

    作用在flex子项上的CSS属性

  • order:改变某一个flex子项的排序位置，所有flex子项默认order属性值为0
  • flex-grow:扩展flex子项所占据的宽度，扩展所侵占的空间就是出去元素外的剩余的空白间隙，默认为0,1为全部空间
  • flex-shrink:属性中的shrink是收缩* flex-shrink主要处理当flex容器空间不足时，每个元素的收缩比例，默认为1
          为0时不收缩
  • flex-basis:定义了在分配剩余空间之前元素的默认大小
  • flex: flex属性是flex-grow，flex-shrink flex-basis的缩写(复合写法),默认为flex:0 1 auto;
  • align-self:只控制单独某一个flex子项的垂直对齐方式,align-items是所有子项

CSSHack（浏览器兼容）

CSS Hack用来解决浏览器的兼容性问题，为不同版本的浏览器定制编写不同的CSS效果

  1. CSS属性前缀法

    前缀标识 兼容浏览器
    _ IE6
    +、* IE6 IE7
    \9(加到值的后面) IE6 IE7 IE8 IE9
    \0(加到值的后面) IE8 IE9 IE10 IE11

  2. 选择器前缀法

    前缀标识 兼容浏览器
    *html IE6
    *+html IE7
    :root IE9及现代浏览器

  3. IE条件注释法(在html中)

    前缀标识 兼容浏览器
    <!--[if\ IE]>...<![endif]--> IE
    <!--[if IE 7]>...<![endif]--> IE7
    <!--[if lte IE 7]>...<![endif]--> IE7 以下
    lte表示less than or equal 也可以gte(greater than or equal)
    <!--[if ! IE 7]>...<![endif]--> 非IE

    line-block可以使元素同时具有block和inline的特征，inline是不支持宽高的 可以使li拥有块的特性又按inline排列，就不需要浮动了

    transition

  • transition-property:规定设置过度效果的CSS属性的名称。
  • transition-duration:规定完成过度效果需要多少秒或毫秒。
  • transition-delay:定义过渡效果何时开始。
  • transition-timing-function:规定速度效果的速度曲线。
    (ease默认值，linear匀速，ease-in加速，ease-out减速，ease-in-out先加速后减速)

在复合写法中第一个时间代表duration,第二个代表delay，其他顺序不影响

transform

transform是css3属性，以下是值

  • translate:位移
  • scale:缩放
  • rotate:旋转(单位是deg(degree)) 等价于rotateZ X和Y作用与3d
  • skew:斜切 单位也是角度，正值向左倾斜
  • transform注意事项：
  1. 变形元素不会影响其他元素(但skew会影响子元素)
  2. 变形元素不能添加给内联元素
  3. 复合写法，可以同时添加多个变形操作，执行是有顺序的，先执行后面的操作,translate会受到scale rotate skew的影响
    (property)trnasform-origin:设置中心基准点

    animation动画

    属性

  • animation-name:设置动画的名字(自定义的)

  • animation-duration:动画的持续时间

  • animation-delay:动画的延迟时间

  • animation-iteration-count:动画的重复次数，默认值就是1,infinite无限次数

  • animation-timing-function:动画的运动形式:

  • linear：动画从头到尾的速度是相同的

  • ease ：动画以低速开始，然后加快，在结束前变慢。默认值

  • ease-in：动画以低速开始

  • ease-out：动画以低速结束

  • ease-in-out：动画以低速开始和结束;

  • animation-full-mode:规定动画播放之前或之后，其动画效果是否可见。

    • 取值：
    1. none(默认值):在运动结束之后回到初始位置，在延迟的情况下，让0%在延迟后生效。
    2. backwards:在延迟的情况下，让0%在延迟前生效。
    3. forwards:在运动结束之后，停到结束位置。
    4. both:backwards和forwards同时生效

  • animation-direction:属性定义是否应该轮流方向播放动画。

  • alternate:一次正向(0%~100%),一次反向(100%-0%)

  • reverse:永远都是反向,100%~0%

  • normal(默认值):永远都是正向,0%~100%*

    在js中动态使用animation

    如果想动态使用可以用dom的document.styleSheets[0]获取css对象，然后使用insertRule方法

    transform3D

    属性

  • rotataX():正值向上翻转

  • rotataY():正值向右翻转

  • translateZ():正值向前，负值向后 Z轴移动 需要先设置 perspective

  • scaleZ():立体元素的厚度 参数是倍数

  • perspective:定义景深

  • transform-style:3D空间 外层加景深，内层加3D空间 flat(2D) preserve-3d(3D空间)

  • transform-origin:center center -50px;(Z轴只能写数值)

  • backface-visibility:隐藏被旋转的 div 元素的背面

    复合写法

  • rotate3d():四个值 0|1(x轴是否添加旋转角度) 0|1（y轴是否添加旋转角度） 0|1(z轴是否添加旋转角度) degree

  • translate3d():三个值 x y z

  • scale3d():三个值 x y z

CSS 模块化

通过sass或less引入文件实现css模块化

文件夹和含义

  • base:一些初始的通用CSS,如重置默认样式，动画，工具，打印等。
  • components:用于勾践页面的所有组件，如按钮，表单，表格，弹窗等。
  • layout:用于布局页面的不同部分，如页眉，页脚，弹性布局，网格布局等。
  • pages:防止页面之间不同的样式，如首页特殊样式，列表页特殊样式等。
  • themes:应用不同的主题样式时，如管理员，买家，卖家等。
  • abstracts:放置一些如:变量，函数，响应式等辅助开发的部分。
  • vendors:防止一些第三方独立的CSS文件，如bootstrap,iconfont等。

CSS 注意事项

控制浏览器大小的css，min-divice-width控制设备大小的样式

padding是控件的内容相对控件的边缘的边距，padding会改变元素的大小．margin是控件边缘的边距．

ul自带margin。

background-position 2个参数时顺序为水平-垂直 而margin是垂直-水平 margin 4个参数顺序为上右下左

body默认宽度是100%(相对于html html是body的父元素)
将html和body的高度同时设置为100%可以使页面和浏览器大小相同
background-position:2个参数 1水平位置 2垂直位置
img有时默认是inline的，可以使用display:block设置成块级元素，或者使用div的background
div中的background会覆盖css文件中的background-xxx，其他的属性也类似，如果要同时在行内选择器和css文件中同时使用最好不要直接用background
margin:auto只在水平方向有效

margin-top的传递问题

子元素的margin-top会把父元素一起带下来 解决： 给父元素加overflow:hidden
