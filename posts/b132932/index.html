安装

  • 先安装node 和 git
  • 在git bash下安装Hexo
  • 将hexo的node_modules文件夹添加环境变量
  • 进入一个空文件夹
  • 初始化hexo
  • 安装npm依赖
  • 生成静态页面

命令

命令 描述
hexo generate 生成静态页面
hexo server 本地预览 调试
hexo deploy 部署
hexo clean 清除缓存

发布

  • hexo clean
  • hexo generate
  • hexo deploy
  • md文档在source/_posts中
  • 在服务器环境需要加-g
  • md文件中的设置冒号后要加空格 否则会生成出错
  • <!--more--> 截断文章，不显示往下的内容
