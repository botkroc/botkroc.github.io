Python

发表于2020-01-20|更新于2020-01-20|python

python中的for in循环 相当于c++里的按置传入 in左侧对象并不是元素的引用
除法运算 (/) 永远返回浮点数类型。如果要做 floor division 得到一个整数结果（忽略小数部分）可以使用 // 运算符,在Python中，可以使用 ** 运算符来计算乘方
在交互模式下，上一次打印出来的表达式被赋值给变量 _
三重引号字符串中的回车换行会自动包含到字符串中，如果不想包含，在行尾添加一个 \ 即可
enumerate()返回一个元组字典,在for循环里使用索引和值赋值这个元组，修改序列需要使用索引，值可以用于打印
值是临时变量，修改序列不影响值
**for循环可以有else：子语句，当循环condition为假时执行，但不会被break执行**
pass 语句什么也不做。当语法上需要一个语句，但程序需要什么动作也不做时，可以使用它
重要警告： 默认值只会执行一次def f(a, L=None):
if L is None:
L = []
在函数调用中，关键字参数必须跟随在位置参数的后面,name 必须出现在 **name 之前
，表示参数必须按字典传递 ,/表示参数必须按顺序传递
仅限位置形参要放在 / (正斜杠) 之前。要将形参标记为 仅限关键字，应在参数列表的第一个 keyword-only 形参之前放置一个 *出现在 *args 参数之后的任何形式参数都是 ‘仅关键字参数’
del var删除整个变量 delvar[:]删除序列中的所有元素，但保留变量
enumerate()返回一个枚举变量，需要转化为list
reversed()和sorted()返回一个副本
Python 与 C 不同，在表达式内部赋值必须显式地使用海象运算符 := 来完成
python里和有多重语义，作为形参时表示任意长度参数，其本质是一个tuple，**表示一个dictionary
作为实参，并且形参为定长时，解引用序列，分别赋值给形参。还作为乘和幂运算符
python默认赋值为引用，将序列副本赋值被其他变量需要用[:]
import module as name 使用name作为模块的别名
for和try，except后面的else表示会在成功时执行(不抛异常视为成功)
python 的 import路径与项目与python本身无关，与系统path相关，可以用sys.path.append()加入路径
