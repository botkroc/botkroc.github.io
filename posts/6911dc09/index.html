DHCP部署与安全

发表2020-12-15|更新2020-12-15|渗透测试

  • DHCP
    Dynamic Host Configuration Protocol

  • DHCP相关概念
    地址池/作用域：（IP、子网掩码、网关、DNS、租期），DHCP使用UDP协议

  • DHCP优点
    减少工作量、避免IP冲突、提高地址利用率

    • 在cisco路由器上面配置DHCP
      ip helper-address 配置DHCP服务器的地址(不需要子网掩码)，此路由器成为DHCP中继

DHCP原理

成为DHCP租约过程，分为4个步骤：

  1. 服务器发送DHCP Discovery广播包
    客户机广播请求IP地址（包括客户机的MAC地址）

  2. 客户机响应DHCP Offer广播包
    服务器响应提供的IP地址（但无子网掩码、网关等参数）

    • 企业级交换机通过限制DHCP Offer广播包来防止伪造DHCP服务器
    • 通过动态绑定MAC地址来防御DHCP攻击

  3. 服务器发送DHCP Request广播包
    客户机选择IP

  4. 服务器发送DHCP ACK广播包(Acknowledgement)
    服务器确定了租约，并提供网卡详细参数IP、掩码、网关、DNS、租约
