关于git push -u命令

发表2021-01-04|更新2021-01-04|开发环境

在首次推送时加上-u参数可以指定本地仓库分支和远程仓库分支的映射关系，以后推送时就不需要指定分支了。

  • 初始化本地仓库
  • 绑定远程仓库

github因为种族歧视将默认master分支更名为main，那么本地默认分支也修改下

  • 修改全局配置更改本地默认分支

git config --global init.defaultBranch main
