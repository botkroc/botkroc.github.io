BOM 简介

Browser Object Model

浏览器对象模型。将整个浏览器看作是一个对象。js中所有成员变量、成员方法都是浏览器对 象的属性和方法。所有的其他对象都是浏览器对象子对象。

BOM 方法

window 对象

方法

  • window.confirm()确认框，通过返回值判断
  • window.prompt()提示框，返回字符串，取消返回null

    属性

  • opener 是一个可读可写的属性，可返回对创建该窗口的 Window 对象的引用。IE内核不支持opener属性
    当使用window.open()打开一个窗口，可以使用此属性返回来自目标窗口源（父）窗口的详细信息。

    location 对象

    location相当于浏览器的地址栏

    方法

  • assign()跳转到指定url
  • reload()重载当前url，如果参数为true，强制加载，从服务器源头重新加载
  • replace()替换当前url 不存在历史记录

    属性

  • location.search:查询字符串 跟在?后面的部分 可读写
  • location.hash:锚点 拥有读写属性

    history 对象

    方法

  • history.back() : 等同于在浏览器点击后退按钮
  • history.forward() : 等同于在浏览器中点击前进按钮
  • history.go() 参数为正前进 为负后退 为0时刷新当前页面

    属性

  • history.length:返回当前页面跳转过的条数
